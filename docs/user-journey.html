When a new user joins the Telegram group, the bot automatically starts an onboarding flow to collect their profile and guide them into the course.

ðŸ’¬ 3. Group Behavior

Commands in Group GROUP

When a user sends any command in the group chat, the bot deletes the command message immediately and sends the response to the user's private DM instead. A brief note ("check your DM") appears in the group for 5 seconds then auto-deletes.

Commands in DM DM

In private chat, commands work normally. Responses appear directly in the conversation with full interactive keyboards.

First-time DM: If a user hasn't started the bot in DM yet, the bot posts a temporary "Start chat with bot" button in the group (auto-deletes after 15s). The user clicks it to open the bot, then commands redirect to DM.

ðŸŽ® 5. Interactive Features

Quiz System

/quiz generates questions from course data with 4 types: term-definition matching (40%), topic identification (25%), level classification (20%), true/false tips (15%). Each question shows the correct answer with explanation. "Next question" button chains quizzes.

Random Tips & Facts

/random pulls a random tip or fact from the pool of 126 tips. Each shows the source topic and level. "Another" button to keep exploring.

Full-Text Search

/search keyword searches across titles, overviews, details, key terms, and tips. Results are scored and ranked by relevance (title matches score highest). Shows top 5 results with snippets.

Topic Navigation

Topic cards include inline buttons: "Key Terms", "Tips", "Back to Level", "Back to Course". Navigation is fully keyboard-driven â€” no need to type commands once inside the course flow.
